# Orbfolio Project Flow

## 1. Visitor Flow

Landing Page
→ Signup/Login

## 2. New User Flow

Signup
→ Choose Username
→ Profile
→ Links
→ Appearance
→ Preview
→ Publish Orbit
→ My Profile

## 3. Existing User Flow

Login
→ My Profile

From My Profile, user can:

- Edit Profile
- Manage Links
- Customize Appearance
- View Analytics
- View Public Orbit
- Publish/Unpublish Orbit
- Change Settings

## 4. Public Orbit Flow

Visitor opens /:username
→ Find Orbit
→ Check publication status
→ Display public Orbit
→ Record profile view
→ Visitor clicks link
→ Record link click
→ Open destination URL
